How do I install angr?

Option 1:

$ mkvirtualenv angr
$ pip install angr

Option 2:

$ docker run -it angr/angr

That's the short answer.

What's the long answer?

angr has a LOT of moving parts and dependencies. If you're on Ubuntu, here are the dependencies you need to install:

$ sudo apt-get install python-dev libffi-dev build-essential virtualenvwrapper

If you're on a wonky platform (read: not in a virtual environment, on windows, etc), pip-install might now work for you. In that case, you should download and install each of the angr subcomponents separately.

Do I have to use a virtual environment?

No, but we'd really, really like you to! We compile and install our own versions of common libraries like z3 and libVEX, and we'd feel bad about installing our versions of those libraries for everyone on your system to use.
